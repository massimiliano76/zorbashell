
## How to Install Zorba CMD
Zorba itself should just work copying the source code using git, just run this commands on GNU/Linux:
sudo -s
cd /root/
git clone https://github.com/zorbaproject/zorba-cmd.git

Anyway, if you want to use advanced features you need to install a few things. You can do it running this command:
/root/zorba-cmd/install.sh
You'll be asked a few questions about what you would actually like to install. If you are not sure, just install everything ansering 'y'.
